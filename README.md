# Анализатор калорийности продуктов

Проект представляет собой десктопное приложение для поиска и анализа пищевой ценности продуктов питания. Разработан в рамках учебного задания по дисциплине программирования.

## Функциональность

- Поиск продуктов по штрихкоду
- Поиск продуктов по названию  
- Отображение основной информации в табличном формате
- Просмотр детальной информации о пищевой ценности
- Интеграция с Open Food Facts API

## Технологии

- Python 3.x
- PyQt5 - для графического интерфейса
- Requests - для работы с HTTP-запросами
- Open Food Facts API - база данных продуктов

## Установка и запуск

### Установка зависимостей
```bash
pip install PyQt5 requests
```
```bash
python main.py
```
